# Coffee House ☕

## Descrição
Projeto de uma landing page para uma cafeteria, desenvolvido para praticar Git e GitHub utilizando Git Flow.

## Tecnologias utilizadas
- HTML5
- CSS3
- Git
- GitHub

## Como executar
1. Clone o repositório.
2. Abra o arquivo `index.html` no navegador.
